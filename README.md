# SESI SENAI GOIÁS

Diretoria de Educação e Tecnologia

Coordenação de Recursos Tecnológicos e Inteligência Educacional

## Produto

Este repositório documenta e deverá evoluir para um produto interno de geração confiável de painéis interativos de inteligência a partir de planilhas Excel.

O fluxo-alvo separa ingestão, validação, normalização, renderização, IA assistiva e publicação:

```text
Excel → validação → normalização → manifesto → componentes versionados → publicação estática
```
